# 🦟 Mata Mosquito

Jogo web em que o jogador precisa "matar" mosquitos que aparecem na tela, testando reflexo e velocidade de resposta.

## 🎯 Objetivo

Projeto desenvolvido durante um curso de Desenvolvimento Web para praticar manipulação do DOM, eventos de clique/interação e lógica de jogo em JavaScript puro.

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## ⚙️ Como rodar localmente

```bash
git clone https://github.com/JVI-JoaoVictor/Mata_Mosquito.git
cd Mata_Mosquito
```

Abra o `index.html` no navegador para começar a jogar.

## 🕹️ Como jogar

- Clique nos mosquitos que aparecem na tela antes que eles fujam
- Acompanhe a pontuação em tempo real
- O jogo tem tela de vitória (`vitoria.html`) e tela de fim de jogo (`fim_de_jogo.html`)

## 💡 Desafios e aprendizados

- Manipulação dinâmica do DOM para criar e remover elementos em tempo real
- Controle de estado do jogo (pontuação, tempo, condições de vitória/derrota)
- Lógica de posicionamento aleatório de elementos na tela
